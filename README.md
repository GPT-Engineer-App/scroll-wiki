# scroll-wiki

A wiki page with different sections. The sections are as follows "background", "childhood", "marriage", "death". The sections can be filled with Lorem Ipsum text. The page should be navigable by a menu on the side, that when clicked automatically scrolls to the correct section.

 Do not use any external npm packages

## Setup

```sh

cd scroll-wiki
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
